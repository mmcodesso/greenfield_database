# Analyze the Data

Once the company story and the business cycles are clear, this section becomes the place where students work with the data. Students can run SQL, rebuild evidence in Excel, read reports, follow guided cases, or browse reusable queries.

For a first assignment, start with SQL Guide, Excel Guide, Reports Hub, or a foundation case. Query Library is useful after students know what kind of question they are trying to answer.

## What Students Do Here

| Subsection | What it is for | Start here |
|---|---|---|
| [SQL Guide](sql-guide.md) | Open SQLite, run one query, and learn how to read SQL output | when the class begins in SQL |
| [Excel Guide](excel-guide.md) | Use the workbook for pivots, filters, charts, and simple reconstruction | when the class begins in Excel |
| [Reports Hub](reports/index.md) | Read finished reports before trying to reproduce them | when students need summary views first |
| [Cases](cases/index.md) | Follow a structured assignment with required student output | when students need guided practice |
| [Query Library](analysis-tracks.md) | Browse reusable SQL by objective, output grain, and source tables | after students know the question they are answering |

## How the Subsections Fit Together

Use this order when students are new to the analytics layer:

1. start with [SQL Guide](sql-guide.md) or [Excel Guide](excel-guide.md) to establish the working method
2. read a report in [Reports Hub](reports/index.md) to see what a finished result looks like
3. use a foundation case in [Cases](cases/index.md) to connect business context, SQL, and interpretation
4. open [Query Library](analysis-tracks.md) when students need more reusable Financial, Managerial, or Audit SQL

## Choosing the First Entry Point

| If the class needs... | Start with |
|---|---|
| query workflow and starter logic | [SQL Guide](sql-guide.md) |
| workbook-based reconstruction | [Excel Guide](excel-guide.md) |
| management-style interpretation | [Reports Hub](reports/index.md) |
| guided assignment flow | [Cases](cases/index.md) |
| reusable query inventory | [Query Library](analysis-tracks.md) |

Each subsection should stay focused on its own job. SQL Guide explains SQL workflow. Excel Guide explains workbook workflow. Query Library organizes reusable SQL. Reports Hub explains reports and perspectives. Cases explains guided assignments.
